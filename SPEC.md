# 火锅外卖 H5 商城的规范

## 项目概述
- **项目名称**: 火锅外卖 H5
- **兼容平台**: 微信、LINE浏览器
- **核心功能**: 商品展示、购物车、下单、配送自取、会员管理、优惠券、库存管理、**多种支付方式**

## 设计风格

### 配色方案
- 主色: #E53935 (麻辣红)
- 辅助色: #FF8A65 (橙红)
- 强调色: #FFD54F (金色)
- 背景色: #FAFAFA (浅灰白)
- 文字色: #333333 (深灰)
- 成功色: #4CAF50 (绿色)

### 字体
- 主字体: -apple-system, BlinkMacSystemFont, 'PingFang SC', 'Hiragino Sans GB', sans-serif

### 尺寸规范
- 设计基准: 375px (iPhone 6/7/8)
- 统一内边距: 12px
- 卡片圆角: 8px
- 按钮圆角: 20px

## 页面结构

### 1. 首页 (index.html)
- 顶部搜索栏
- 轮播Banner
- 快捷分类入口
- 热门商品列表
- 底部Tab导航

### 2. 分类页 (category.html)
- 左侧分类列表
- 右侧商品展示

### 3. 购物车页 (cart.html)
- 商品列表
- 数量调整
- 优惠券选择
- 价格汇总

### 4. 商品详情页 (product.html)
- 商品图片
- 价格和规格
- 加入购物车

### 5. 下单页 (checkout.html)
- 配送/自取切换
- 地址填写
- 优惠券选择
- 订单确认

### 6. 会员中心 (member.html)
- 会员信息
- 订单记录
- 优惠券管理
- 收货地址

### 7. 库存管理页 (admin.html)
- 商品列表
- 库存修改
- 上下架管理

## 支付方式
1. 💚 **微信支付** - 微信内支付首选
2. 💙 **支付宝** - 支付宝安全支付
3. 💛 **PayPay** - 日本地区常用
4. 💚 **LINE Pay** - LINE生态支付

### 支付流程
1. 用户选择支付方式
2. 生成支付二维码
3. 用户扫码完成支付
4. 支付回调确认
5. 订单状态更新

## 底部Tab导航
1. 首页
2. 分类
3. 购物车
4. 会员

## 技术实现
- 纯HTML + CSS + JavaScript
- LocalStorage 数据持久化
- 响应式设计
- 微信/LINE 兼容
